# Commans installation
```
git clone https://github.com/RaulXH/ShortXH && cd ShortXH
chmod +x ShortXH && ./ShortXH
```
# Boot from the PATH
```
wget https://git.io/JB84e -O $PREFIX/bin/ShortXH

ShortXH -h
```
